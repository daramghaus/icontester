Icon Test Tool for Designer 디자이너를 위한 아이콘 테스트 툴 Icon Tester is Web App, you will be able to test easily your icon by Add to Home Screen function on iOS - Safari or Android - Google Chrome.
To use Icon Tester you will need your Web Server(Local or Internet). Icon Tester는 iOS - Safari 또는 Android - Google Chrome의 홈 화면에 추가 기능을 사용하여 쉽게 아이콘을 테스트할 수 있는 웹 앱입니다.
Icon Tester를 쓰려면 당신의 웹 서버(로컬 또는 인터넷)가 필요합니다. Icon Tester Ver.2.2 Icon Tester Ver.2.2
How to Use 사용법
  1. Download Icon Tester to your Mac or PC.
  2. Unzip and Upload all files(index.html, css and images folders) to your Web Server.
  3. Overwrite your icon image files on images folder.
    Android: Overwrite "images/android/icon.png" image.
  4. Open the index.html on device default web browser of your device.
    iOS: Safari(iPhone or iPad)
    Android: Google Chrome
  5. Add to Home Screen.
    iOS: Tap Share bottun and then Add to Home Screen
    Android: Tap menu button(top right), Tap Add to Home Screen.
  6. If you want Icon refresh, Overwrite icon image files again,
    iOS: Run and close the icon on Home Screen, Refresh automatically.
    Android: Delete Icon on Home Screen, and add icon again on Chrome.
TIP You can compare multiple icons. Create multiple Icon Tester folders on the Web server, and "Add to Home Screen" on each web page.
  1. Icon Tester를 맥 또는 PC에 다운로드합니다.
  2. 압축을 풀고, 모든 파일(index.html, css, images 폴더)을 당신의 웹 서버에 업로드합니다.
  3. 작업한 아이콘 이미지 파일을 "images"폴더에 덮어씁니다.
    Android에서는 "images/android/icon.png" 이미지만 교체하면 "홈 화면"에 보입니다. 디바이스의 해상도와 상관없이 해당 이미지만 교체해서 테스트하세요.
  4. index.html를 디바이스의 기본 웹 브라우저에서 엽니다.
    iOS: Safari(iPhone or iPad)
    Android: Google Chrome
  5. 홈 화면에 아이콘을 추가합니다.
    iOS: 공유하기 버튼을 탭한 다음, 홈 화면에 추가합니다.
    Android: Chrome에서 우측 상단 메뉴버튼을 눌러, 홈 화면에 추가를 선택합니다.
  6. 아이콘을 갱신하려면, 아이콘 이미지 파일을 다시 덮어 씌우고,
    iOS: 홈 화면에 있는 아이콘을 실행하고 닫으면 자동으로 아이콘이 바뀝니다. Android: 아이콘을 삭제하고 다시 추가합니다.
TIP 여러 아이콘을 비교할 수 있습니다. 웹 서버에 여러 Icon Tester 폴더를 만들고, 각각의 웹페이지에서 "홈 화면에 추가"하세요.
